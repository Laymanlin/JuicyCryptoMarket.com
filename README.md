## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager
- Modern web browser

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Laymanlin/JuicyCryptoMarket.com.git
cd JuicyCryptoMarket.com
```

2. Install dependencies:
```bash
npm install
```

3. Create environment configuration:
```bash
cp .env.example .env
```

4. Configure your environment variables in `.env`

5. Start the application:
```bash
npm start
```

6. Open your browser and navigate to `http://localhost:3000`

## Usage

### Creating an Account

1. Click on "Sign Up" button
2. Enter your email and create a password
3. Verify your email address
4. Complete KYC verification

### Trading Cryptocurrencies

1. Log in to your account
2. Browse available cryptocurrencies
3. Select a cryptocurrency to trade
4. Enter the amount and confirm transaction
5. Monitor your portfolio in the dashboard
